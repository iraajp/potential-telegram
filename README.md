# Minimal Personal Website

Ultra-clean personal website inspired by Aman Sanger's design aesthetic.

## Features

- Dark theme with high whitespace
- Large typography with Inter font
- Left-aligned content with right-aligned metadata
- Books page with star ratings and notes
- Thoughts page with categorized content
- Fully responsive design
- Zero animations, borders, or shadows

## Tech Stack

- React 18
- React Router 6
- Tailwind CSS 3
- Vite

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Visit `http://localhost:5173`

## Build

```bash
npm run build
```
